# 📘 MongoDB Book Collection Scripts

This project contains MongoDB scripts to perform various operations on a `books` collection. It includes basic queries, advanced filtering, aggregation pipelines, and indexing for performance.

---

## 🚀 Requirements

- MongoDB installed locally or connected via MongoDB Atlas
- Mongo Shell (`mongosh`) or VS Code with **MongoDB for VS Code** extension
- JSON/JS file with script content (e.g., `books_script.js`)
